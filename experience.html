I was able to take what I learned from Python and apply it to other languages which made the learning process a lot easier. I also have experience in object oriented languages like C# and Java. Knowing all of these different programming languages allows me to work on many different kinds of projects.

Database/Data Science

In this class I learned many things like how to find specific data in a large dataset and display it, how to cleanse missing data, and even a little bit of machine learning. This was all so cool to experience seeing the power of data and how it can be applied to real world problems. I think that going into the data science or database industry could be a very high possibility in the future.
